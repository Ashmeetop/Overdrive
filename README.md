## Installation

1. Make sure you have [Fabric Loader](https://fabricmc.net/use/) installed.
2. Download the latest **Overdrive_2** `.jar` from [Releases](https://github.com/ASHMEET/Overdrive_2/releases).
3. Place the `.jar` into your Minecraft `mods` folder.
4. Launch Minecraft with the Fabric profile.

## Usage

- The mod works automatically upon launching the game.
- Advanced settings can be modified in the `config` folder.
